# Ejercicio 05 - JavaScript

## Introducción
**JavaScript** es el lenguaje de programación que dota de vida e interactividad a la web. Mientras que HTML define la estructura y CSS el estilo, JavaScript permite que los elementos de una página respondan a las acciones del usuario, procesen datos y se comuniquen con servidores. En el ecosistema del desarrollo moderno, es una herramienta indispensable que ha evolucionado de ser un simple lenguaje de scripts para el navegador a una plataforma robusta capaz de ejecutar aplicaciones complejas en diversos entornos.

## Síntesis
JavaScript es un lenguaje de programación de **alto nivel, dinámico e interpretado**. Una de sus características más importantes, discutida en clase, es su capacidad para manipular el DOM (Document Object Model), lo que permite cambiar el contenido de una página sin necesidad de recargarla. El lenguaje maneja diversos tipos de datos, desde primitivos como números y cadenas de texto (strings), hasta estructuras más complejas como objetos y arreglos (arrays), permitiendo organizar la información de manera lógica y accesible para el programador.

El flujo de una aplicación en JavaScript se controla mediante estructuras fundamentales como los condicionales (`if`, `else`, `switch`) y los ciclos o bucles (`for`, `while`). Estas herramientas permiten que el código tome decisiones basadas en variables o eventos. Otro pilar esencial es el concepto de **funciones**, que son bloques de código reutilizables diseñados para realizar tareas específicas. En la actualidad, el estándar ECMAScript ha introducido mejoras significativas, como las funciones de flecha (arrow functions) y las promesas, que facilitan el manejo de operaciones asíncronas, como las peticiones a APIs externas.

Finalmente, JavaScript se distingue por su modelo de programación basado en **eventos**. Esto significa que el código puede "escuchar" acciones del usuario, como clics de ratón, pulsaciones de teclas o el desplazamiento de la página, y ejecutar una respuesta inmediata. Esta naturaleza asíncrona y reactiva es lo que permite crear experiencias de usuario fluidas y dinámicas, convirtiendo documentos estáticos en aplicaciones web interactivas y potentes que forman la base de la navegación actual.

## Reflexión
Lo que considero más útil de JavaScript es su capacidad para convertir una página web en una **herramienta funcional**. Me parece impresionante cómo unas pocas líneas de código pueden cambiar por completo la experiencia de un usuario, desde validar un simple formulario hasta crear animaciones complejas o juegos en el navegador. La lógica detrás de la programación me ayuda a pensar de forma más estructurada y me entusiasma la idea de que, con este lenguaje, no hay límites para lo que se puede construir en la web.

## Conclusión
En conclusión, JavaScript es el motor que impulsa la interactividad en la red. A través de este ejercicio, se ha explorado cómo su sintaxis y su lógica de eventos permiten transformar el contenido estático en experiencias dinámicas. Su integración con HTML y CSS completa la tríada fundamental del desarrollo web, demostrando que el dominio de JavaScript es el paso definitivo para pasar de ser un diseñador de páginas a un desarrollador de aplicaciones web completas.
